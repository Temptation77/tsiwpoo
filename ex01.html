EXERCÍCIO Nº1

1) Crie um ficheiro (ex01.js) que inclua uma função que receba uma string e que retorne uma nova string cifrada.
A cifra é simples: a nova string será composta, alternadamente, pelas extremidades na string original até que já não exista qualquer caracter para processar.

Todos os dados de input e output devem ser obtidos/colocados neste ficheiro HTML que não deve ser alterado.

String não cifrada: String cifrada:

Teste o seu código para os seguintes casos de teste:
LA -> LA
AMOR -> ARMO
ESMAD -> EDSAM
